# Checklist Registrazione – EP04 “Perché sono diventato vegan”

## Prima della registrazione
- [ ] Microfono cardioide + pop‑filter, gain 30–40 %
- [ ] Stanza trattata / rumore < −50 dB (registra 10 s di silenzio)
- [ ] Script aperto su tablet con marcatori di scena (1–5)
- [ ] Test frase: “Il nostro corpo produce tutto il colesterolo…” (peak ≤ −6 dB)

## Durante la registrazione
- [ ] Mantieni distanza 20–25 cm; postura eretta
- [ ] Take separati per ogni scena (Intro, C2, C3, C4, Outro)
- [ ] Evita consonanti esplosive (“p”, “t”) – angola leggermente il microfono
- [ ] Annota timecode degli errori per punch‑in

## Post‑produzione audio
- [ ] Noise print dai 10 s iniziali → riduzione delicata
- [ ] EQ: low‑cut 80–90 Hz; +2 dB @ 5 kHz per intelligibilità
- [ ] Compressione leggera (ratio 2:1, attacco 20 ms, rilascio 120 ms)
- [ ] Normalizzazione a −14 LUFS integrato; TP ≤ −1 dBFS
- [ ] Esporta `voice_EP06_vegan.wav` (48 kHz / 24 bit)
