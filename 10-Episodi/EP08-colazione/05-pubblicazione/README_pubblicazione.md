# EP01 — Pubblicazione

Checklist e materiali dedicati alla pubblicazione.

- **EP01-colazione_checklist_pubblicazione.md**
- **meta_EP01_title_description_tags.md** (da creare)
- **PUBLISHED** (solo dopo upload YouTube)

## Workflow
1. Preparare titolo, descrizione, tags e thumbnail.
2. Caricare il video finalizzato.
3. Creare il file “PUBLISHED” con URL YouTube.
4. Aggiornare l’indice generale (Index_Episodi.md).
