# EP01 — Script

Questa cartella contiene tutti gli script narrativi di EP01, in ordine di lavoro:

- **EP01-colazione.md**  
  Versione originale / contenuto grezzo.

- **EP01-colazione_script_voce.md**  
  Script definitivo per la registrazione voce.

- **EP01-breakfast_EN.md**  
  Traduzione inglese completa.

- **EP01-breakfast_pronunciation.md**  
  Note di pronuncia per la versione inglese.

## Workflow
1. Lavorare sempre su “EP01-colazione_script_voce.md”.
2. Aggiornare la versione inglese solo dopo la versione finale italiana.
3. Annotare modifiche o dubbi in questo README.
