# EP01 — Colazione

Vegan Biohacking Routine · Episodio 01

Questo episodio introduce la routine mattutina vegana e la preparazione della colazione funzionale.

## Struttura episodio

- **01-script/** → script finali e traduzioni
- **02-storyboard/** → storyboard e coreografia
- **03-registrazione/** → checklist e setup luci
- **04-montaggio/** → timeline, doppiaggio, checklist
- **05-pubblicazione/** → metadata e pubblicazione
- **assets/** → materiali video/foto
- **notes/** → note grezze di lavorazione

## Stato produzione

Consulta `Index_Episodi.md` per lo stato globale.

## Workflow consigliato

1. Preparazione → luce + checklist
2. Registrazione → riprese + audio
3. Montaggio → timeline + doppiaggio
4. Export → master finale
5. Pubblicazione → metadata + upload
